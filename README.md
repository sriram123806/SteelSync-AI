# SteelSync-AI

SteelSync AI is a multi-agent logistics orchestration platform built for the Band of Agents Hackathon. The system demonstrates how specialized AI agents can communicate and collaborate to automate logistics workflows. The platform consists of six independent agents: Commander Agent, ERP Validator Agent, Router Agent, Inventory Agent, Demurrage Agent, and Scenario Agent. Instead of relying on a single AI model, each agent is responsible for a specific logistics function and communicates through the Band platform. The workflow begins when a shipment request is received. The ERP Validator Agent verifies shipment information and checks for missing or invalid fields. The Router Agent determines the transportation mode. The Inventory Agent checks stock availability and resource readiness. The Demurrage Agent estimates potential storage and detention risks. Finally, the Scenario Agent generates best-case, expected-case, and worst-case outcomes for the shipment. The Commander Agent orchestrates the complete workflow, tracks progress, receives outputs from all agents, and generates the final logistics report. The project demonstrates agent-to-agent communication, workflow coordination, task delegation, distributed decision-making, and multi-agent collaboration using Band of Agents, Gemini, LangChain, and LangGraph. SteelSync provides a foundation for future logistics automation systems where multiple intelligent agents cooperate to improve shipment planning, inventory visibility, operational efficiency, and supply chain decision support.
